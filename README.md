# autonomous-drone-platform
saronic im coming for u

A Python-based command-line controller for ArduPilot SITL using DroneKit and MAVLink.

Current features:
- Connects to ArduCopter SITL
- Saves home location
- Takes off to user-specified altitude
- Flies to GPS waypoint
- Returns home
- Lands and confirms successful landing
- Displays vehicle status
